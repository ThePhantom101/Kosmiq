# Workflow: New Component

Steps:
1. Read ROUTES.md and confirm where this component lives
2. Check graphify-out/graph_report.md for existing similar components
3. Use @react-patterns — scaffold component with TypeScript props interface
4. Use @tailwind-design-system — apply only tokens from tokens.css
5. Apply @wcag-audit-patterns — check keyboard nav and aria labels
6. Use @code-review — audit before finishing
7. List any new dependencies required before installing anything
